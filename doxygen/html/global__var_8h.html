scan_info_t* return_ap_in_range ( int  wiphy)

returns a list of aps detected by "wiphy" interface using scan_info_t struct. don't fill all fields.
fills only: ap_num, wiphy, intf_name, mac_addr, beacon_interval, capability, channel, is_dBm, signal, age, num_rates, rates

free_scan_info_t(aps);
